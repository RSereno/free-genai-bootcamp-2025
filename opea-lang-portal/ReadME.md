## Start virtual enviroment

``` sh
curl -LsSf https://astral.sh/uv/install.sh | sh

uv venv opean_lang --python 3.12 --seed

source opean_lang/bin/activate
```

Reference: https://docs.astral.sh/uv/#getting-started
